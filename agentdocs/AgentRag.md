We stream live market data from Polymarket and live fight statistics from the balldontlie MMA API into a unified event feed. When a meaningful change occurs, the agent gathers recent context and relevant historical patterns, then decides what information matters. Using retrieval-augmented generation, it pulls supporting evidence, reasons about possible causes, and produces a short natural-language explanation. A lightweight verification step checks the explanation against available data; if confidence is low, it returns a neutral response instead of guessing. The explanation and references are sent to the UI in real time. The system stays modular so the agent can adapt its retrieval strategy, choose tools dynamically, and improve reasoning over time without changing the core pipeline.
